-   Opcode: **0x092**
-   Short name: **PCTURN**
-   Long name: Turn to face PC

#### Argument

none

#### Stack

  
*(?)*

*Speed*

**PCTURN**

#### Description

Make this entity face the PC. *Speed* is number of frames (larger =
slower turn).
