-   Opcode: **0x020**
-   Short name: **IDUNLOCK**
-   Long name: Unlock walkmesh ID

#### Argument

Walkmesh triangle ID

#### Stack

none

#### Description

Unlocks a walkmesh triangle so things can walk over it.
