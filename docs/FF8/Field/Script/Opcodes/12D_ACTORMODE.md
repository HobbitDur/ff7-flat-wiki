---
title: 12D_ACTORMODE
---

-   Opcode: **0x12D**
-   Short name: **ACTORMODE**
-   Long name: Set Actor Mode

#### Argument

none

#### Stack

  
*Mode*

**ACTORMODE**

#### Description

Controls transparency/blinking effects on models (might also control whether Squall/Seifer's gunblades are visible).
