-   Opcode: **0x155**
-   Short name: **SETDRAWPOINT**
-   Long name: Set Draw Point hidden

#### Argument

isHidden

#### Stack

  
Bool (PSHN\_L)

**SETDRAWPOINT**

#### Description

If isHidden is bigger or equal to 1, then hides draw point. If not, the
draw point is visible.
