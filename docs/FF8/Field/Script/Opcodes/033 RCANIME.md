-   Opcode: **0x033**
-   Short name: **RCANIME**
-   Long name: Resume script, controlled animation

#### Argument

Model Animation ID

#### Stack

  
*Last frame of animation*

*First frame of animation*

**RCANIME**

#### Description

Play the range of frames of an animation, then returns the model to its
base animation.
