-   Opcode: **0x0126**
-   Short name: **SETWITCH**
-   Long name: Set Rinoa as Sorceress

#### Argument

none

#### Stack

  
*1*

**SETWITCH**

#### Description

This is only called once in the whole game - the first time you enter
the Ragnarok in space. It unlocks Rinoa's alternate limit break after
she becomes a sorceress.
