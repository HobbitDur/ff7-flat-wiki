-   Opcode: **0x049**
-   Short name: **MESVAR**
-   Long name: Set Message Variable

#### Argument

none

#### Stack

  
*Variable number*

*Value*

**MESVAR**

#### Description

Sets the value of a message variable. Message variables are used in
texts. In the Deling field editor, message variables can be placed in
text using {Var0}, {Var1}, etc.
