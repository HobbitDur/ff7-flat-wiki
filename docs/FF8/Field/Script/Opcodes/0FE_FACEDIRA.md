---
title: 0FE_FACEDIRA
---

-   Opcode: **0x0FE**
-   Short name: **FACEDIRA**
-   Long name: Head Face towards Actor

#### Argument

none

#### Stack

  
*Actor code (use PSHAC)*

*Frame count*

**FACEDIRA**

#### Description

Turns this actor's head to face the given actor over the duration of *frame count*.
