-   Opcode: **0x02E**
-   Short name: **ANIMEKEEP**
-   Long name: Animate, keep last frame

#### Argument

Model Animation ID

#### Stack

none

#### Description

Play an animation, then freeze the model on the last frame of the
animation. Pauses script until animation finishes.
