-   Opcode: **0x116**
-   Short name: **RAMESW**
-   Long name: Resume script, A? Message, and Wait

#### Argument

none

#### Stack

  
*Message Channel*

*Field message ID*

*X position of window*

*Y position of window*

**RAMESW**

#### Description

Pop up a message window and wait for the player to hit "ok." Unlike
[AMESW][], RAMESW will let the script continue running.
