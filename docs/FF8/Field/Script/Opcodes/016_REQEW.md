---
title: 016_REQEW
---

-   Opcode: **0x016**
-   Short name: **REQEW**
-   Long name: Request remote execution (synchronous, guaranteed)

#### Argument

The ID of the target Entity.

#### Stack

..., *Priority*, *Label* =&gt; ...

#### Description

Go to the method *Label* in the group **Argument** with a specified *Priority*.
