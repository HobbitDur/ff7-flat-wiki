-   Opcode: **0x179**
-   Short name: **UNKNOWN14**
-   Long name: Preserve Sound Channel

#### Argument

none

#### Stack

  
*Sound channel*

**UNKNOWN14**

#### Description

Prevents the given sound channel from being silenced when a new field is
loaded. The currently playing sound effect will continue to play during
the fade.

Does not work on channel 0.
