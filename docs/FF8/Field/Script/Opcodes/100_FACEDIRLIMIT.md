title: 100_FACEDIRLIMIT
---

-   Opcode: **0x100**
-   Short name: **FACEDIRLIMIT**
-   Long name: Limit Head Facing

#### Argument

none

#### Stack

  
*Angle 1?*

*Angle 2?*

*Angle 3?*

**FACEDIRLIMIT**

#### Description

Limits how far this actor's head can turn to face an actor (so they don't go into exorcist mode).
