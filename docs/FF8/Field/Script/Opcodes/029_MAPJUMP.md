title: 029_MAPJUMP
---

-   Opcode: **0x029**
-   Short name: **MAPJUMP**
-   Long name: Jump to map

#### Argument

Walkmesh ID

#### Stack

  
*Field Map ID*

*XCoord*

*YCoord*

*ZCoord*

**MAPJUMP**

#### Description

Jump the player to the field with the given ID at the specified coordinates.
