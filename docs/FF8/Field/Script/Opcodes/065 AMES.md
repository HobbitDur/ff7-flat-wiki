-   Opcode: **0x065**
-   Short name: **AMES**
-   Long name: A? Message

#### Argument

none

#### Stack

  
*Message Channel*

*Field message ID*

*X position of window*

*Y position of window*

**AMES**

#### Description

Pop up a message window until [WINCLOSE][] or [MESSYNC][] is called.
