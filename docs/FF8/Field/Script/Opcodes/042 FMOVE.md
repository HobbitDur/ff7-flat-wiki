-   Opcode: **0x042**
-   Short name: **FMOVE**
-   Long name: Facing? Move

#### Argument

none

#### Stack

  
*X*

*Y*

*Z*

*????* (always 0?)

**FMOVE**

#### Description

Moves this entity. Unlike the other move functions, this will not cause
the entity's walk animation to play, but will make it face in the
direction of movement.
