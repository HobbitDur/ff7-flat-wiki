---
title: 153_ADDSEEDLEVEL
---

-   Opcode: **0x153**
-   Short name: **ADDSEEDLEVEL**
-   Long name: Add SeeD Level

#### Argument

none

#### Stack

  
Amount

**ADDSEEDLEVEL**

#### Description

Add the given amount of points to the player's SeeD level.

100 points per rank - minimum 100 (rank 1), maximum 3100 (rank A)
