-   Opcode: **0x00D**
-   Short name: **POPM\_W**
-   Long name: Pop to memory (word)

#### Argument

Memory adress.

#### Stack

..., *value* =&gt; ...

#### Description

Pop *value* from stack and store the first two bytes (word) in
**Argument**.
