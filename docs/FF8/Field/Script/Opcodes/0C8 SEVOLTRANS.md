-   Opcode: **0x0C8**
-   Short name: **SEVOLTRANS**
-   Long name: Sound Effect Volume Transition

#### Argument

none

#### Stack

  
*Sound channel*

*Frame count*

*Final volume*

**SEVOLTRANS**

#### Description

Transitions the volume of the sound in *sound channel* over *frame
count*.
