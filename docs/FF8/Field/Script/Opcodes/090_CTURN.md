-   Opcode: **0x090**
-   Short name: **CTURN**
-   Long name: Turn to face character

#### Argument

none

#### Stack

  
*Entity ID to face*

*Duration of turn (frames)*

**CTURN**

#### Description

Make this entity face the entity with the ID of the first parameter. That parameter is always set with [PUSHAC](013_PSHAC.md).
