-   Opcode: **0x065**
-   Short name: **AMES**
-   Long name: A? Message

#### Argument

none

#### Stack

  
*Message Channel*

*Field message ID*

*X position of window*

*Y position of window*

#### Description

Pop up a message window until [WINCLOSE](FF8/Field/Script/Opcodes/04C_WINCLOSE "wikilink") or [MESSYNC](048_MESSYNC.md) is called.
