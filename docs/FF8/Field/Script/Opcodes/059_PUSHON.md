---
title: 059_PUSHON
---

-   Opcode: **0x059**
-   Short name: **PUSHON**
-   Long name: Push script on

#### Argument

none

#### Stack

none

#### Description

Enables "pushing" this entity. An entity's "push" script is run when the player walks into it. Detection range can presumably be set with [PUSHRADIUS](063_PUSHRADIUS.md)
