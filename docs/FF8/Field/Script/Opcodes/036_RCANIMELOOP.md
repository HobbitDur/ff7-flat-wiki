---
title: 036_RCANIMELOOP
---

-   Opcode: **0x036**
-   Short name: **RCANIMELOOP**
-   Long name: Resume script, Play controlled looping animation

#### Argument

Model Animation ID

#### Stack

  
*Last frame of animation*

*First frame of animation*

**RCANIMELOOP**

#### Description

Loops the given frames of an animation.
