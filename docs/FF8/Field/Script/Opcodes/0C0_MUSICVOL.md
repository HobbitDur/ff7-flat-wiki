---
title: 0C0_MUSICVOL
---

-   Opcode: **0x0C0**
-   Short name: **MUSICVOL**
-   Long name: Set Music Volume

#### Argument

none

#### Stack

  
*0 or 1*

*Volume (0-127)*

**MUSICVOL**

#### Description

Set the volume of the music. All the music functions have a parameter that's either 0 or 1.
