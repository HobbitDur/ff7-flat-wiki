-   Opcode: **0x04E**
-   Short name: **UCOFF**
-   Long name: User Control Off

#### Argument

none

#### Stack

none

#### Description

Disable user control. Player will only be able to press "ok" (and
pause?) to advance through dialogue. Ends when [UCON][] is called.
