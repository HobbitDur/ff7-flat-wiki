-   Opcode: **0x0C9**
-   Short name: **SEPOS**
-   Long name: Set Sound Effect Pan

#### Argument

none

#### Stack

  
*Sound channel*

*Pan (0-255)*

**SEPOS**

#### Description

Sets the pan of the sound playing in *sound channel*. 0=left, 255=right.
