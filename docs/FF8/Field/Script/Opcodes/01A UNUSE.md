-   Opcode: **0x01A**
-   Short name: **UNUSE**
-   Long name: Unuse entity

#### Argument

Always 0.

#### Stack

No change.

#### Description

Disable this entity's scripts, hides its model, and makes it
throughable. Call [USE][] to re-enable.
