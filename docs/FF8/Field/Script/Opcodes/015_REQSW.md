---
title: 015_REQSW
---

-   Opcode: **0x015**
-   Short name: **REQSW**
-   Long name: Request remote execution (asynchronous execution, guaranteed)

#### Argument

The ID of the target Entity.

#### Stack

..., *Priority*, *Label* =&gt; ...

#### Description

Go to the method *Label* in the group **Argument** with a specified *Priority*.
