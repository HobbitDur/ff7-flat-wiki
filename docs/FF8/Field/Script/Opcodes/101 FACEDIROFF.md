-   Opcode: **0x101**
-   Short name: **FACEDIROFF**
-   Long name: Stop head facing

#### Argument

none

#### Stack

  
*Frame count*

**FACEDIROFF**

#### Description

End the current FACEDIR over the course of *frame count*, then locks the
head back on the actor.
