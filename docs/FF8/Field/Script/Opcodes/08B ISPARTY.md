-   Opcode: **0x08B**
-   Short name: **ISPARTY**
-   Long name: Get position in party

#### Argument

none

#### Stack

  
*Character ID*

**ISPARTY**

#### Description

Pushes the character's position in the party list into temporary
variable 0. If they're not in the party, pushes -1.
