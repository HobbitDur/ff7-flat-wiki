-   Opcode: **0x01E**
-   Short name: **SET3**
-   Long name: Set 3-coordinate position

#### Argument

Walkmesh triangle ID.

#### Stack

  
*XCoord*

*YCoord*

*ZCoord*

**SET3**

#### Description

Place this entity's model at XCoord, YCoord, ZCoord standing on the
given walkmesh triangle.
