-   Opcode: **0x00F**
-   Short name: **POPM\_L**
-   Long name: Pop to memory (long)

#### Argument

Memory adress.

#### Stack

..., *value* =&gt; ...

#### Description

Pop *value* from stack and store the first four bytes (long) in
**Argument**.
