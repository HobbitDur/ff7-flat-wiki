-   Opcode: **0x01F**
-   Short name: **IDLOCK**
-   Long name: Lock walkmesh ID

#### Argument

Walkmesh triangle ID

#### Stack

none

#### Description

Locks a walkmesh triangle so nothing can walk over it.
