-   Opcode: **0x0c0**
-   Name: push logicor
-   Two-word opcode: No

#### Parameters

| Location | Name |  Description  |
|:--------:|:----:|:-------------:|
|  Stack   |  A   | Boolean value |
|  Stack   |  B   | Boolean value |

#### Notes

Pushes 0 to the stack if both A and B are 0, otherwise 1.
