-   Opcode: **0x050**
-   Name: push shl
-   Two-word opcode: No

#### Parameters

| Location | Name |    Description     |
|:--------:|:----:|:------------------:|
|  Stack   |  A   |   Value to shift   |
|  Stack   |  B   | Positions to shift |

#### Notes

Pushes A &lt;&lt; B to the stack.
