-   Opcode: **0x324**
-   Name: set window dimensions
-   Two-word opcode: No

#### Parameters

| Location | Name | Description  |
|:--------:|:----:|:------------:|
|  Stack   |  X   | X coordinate |
|  Stack   |  Y   | Y coordinate |
|  Stack   |  W   |    Width     |
|  Stack   |  H   |    Height    |

#### Notes

Works similarly to the WINDOW field opcode, except there is only one window so no ID is necessary.
