-   Opcode: **0x118**
-   Name: push byte from bank0
-   Two-word opcode: Yes

#### Parameters

| Location |  Name   |     Description      |
|:--------:|:-------:|:--------------------:|
|   Code   | Address | Byte address in bank |

#### Notes

Pushes the byte at Address from bank0 (Bank 1 in the [Savemap][]).

This is a valid parameter to the [write bank opcode][].
