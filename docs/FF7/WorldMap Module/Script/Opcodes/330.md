-   Opcode: **0x330**
-   Name: set active entity
-   Two-word opcode: No

#### Parameters

| Location |   Name   |          Description          |
|:--------:|:--------:|:-----------------------------:|
|  Stack   | Model ID | Model to set as active entity |

#### Notes

Sets the current active entity to the entity representing the specified
model.
