-   Opcode: **0x316**
-   Name: set point sky bottom BGR
-   Two-word opcode: No

#### Parameters

| Location | Name | Description |
|:--------:|:----:|:-----------:|
|  Stack   |  B   |  Blue tint  |
|  Stack   |  G   | Green tint  |
|  Stack   |  R   |  Red tint   |

#### Notes

Light effect to be applied to the bottom portion of sky in the area
affected by this light source.
