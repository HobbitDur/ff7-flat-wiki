-   Opcode: **0x051**
-   Name: push shr
-   Two-word opcode: No

#### Parameters

| Location | Name |    Description     |
|:--------:|:----:|:------------------:|
|  Stack   |  A   |   Value to shift   |
|  Stack   |  B   | Positions to shift |

#### Notes

Pushes A &gt;&gt; B to the stack.
