-   Opcode: **0x201**
-   Name: jump if false
-   Two-word opcode: Yes

#### Parameters

| Location |   Name    |          Description           |
|:--------:|:---------:|:------------------------------:|
|   Code   |  Address  | Destination address to jump to |
|  Stack   | Condition |         Boolean value          |

#### Notes

Jumps to the address specified if Condition is false.
