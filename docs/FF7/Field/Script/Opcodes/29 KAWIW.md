-   Opcode: **0x29**
-   Short name: **KAWIW**
-   Long name: Wait For Kawai

#### Memory layout

| 0x29 |
|------|

#### Arguments

None.

#### Description

Halts script execution until a prior [KAWAI](FF7/Field/Script/Opcodes/28_KAWAI "wikilink") operation has completed, such as a [LIGHT](28 KAWAI/06 LIGHT.md) movement.
