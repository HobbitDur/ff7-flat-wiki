-   Opcode: **0x54**
-   Short name: **WCLSE**
-   Long name: Window Close

#### Memory layout

| 0x54 | *N* |
|------|-----|

#### Arguments

-   **const UByte** *N*: The ID of the window to close.

#### Description

Closes the given window.
