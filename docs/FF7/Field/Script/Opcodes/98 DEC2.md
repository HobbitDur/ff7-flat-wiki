-   Opcode: **0x98**
-   Short name: **DEC2**
-   Long name: Decrement (16-bit)

#### Memory layout

| 0x98 | *B* | *A* |
|------|-----|-----|

#### Arguments

-   **const UByte** *B*: Destination bank.
-   **const UByte** *A*: Address.

#### Description

Decrements the 16-bit value found at bank **B**, address **A**. If the
value is 0x0000, it will roll over to 0xFFFF.
