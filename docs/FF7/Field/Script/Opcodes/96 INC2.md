-   Opcode: **0x96**
-   Short name: **INC2**
-   Long name: Increment (16-bit)

#### Memory layout

| 0x96 | *B* | *A* |
|------|-----|-----|

#### Arguments

-   **const UByte** *B*: Destination bank.
-   **const UByte** *A*: Address.

#### Description

Increments the 16-bit value found at bank **B**, address **A**. If the
value is 0xFFFF, it will roll over to 0x0000.
