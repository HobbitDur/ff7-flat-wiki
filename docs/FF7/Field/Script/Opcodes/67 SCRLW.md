-   Opcode: **0x67**
-   Short name: **SCRLW**
-   Long name: Wait for scroll

#### Memory layout

| 0x67 |
|------|

#### Arguments

None.

#### Description

Halts execution of the current script for a preceding scroll to have
reached its destination coordinates before continuing.
