-   Opcode: **0xFF**
-   Short name: **GAMEOVER**
-   Long name: Game over

#### Memory layout

| 0xFF |
|------|

#### Arguments

None.

#### Description

Instantaneously ends the current game, displaying the 'Game Over'
screen; subsequent script commands are not executed. As per a normal
game over, after the user presses \[OK\], the game returns to the main
start menu.
