-   Opcode: **0xDD**
-   Short name: **ANIMB**
-   Long name: Stop Animation.

#### Memory layout

| 0xDD |
|------|

#### Arguments

None.

#### Description

Stop current playing animation in the current frame (set current frame
as last frame). Model will stay in this last frame until next animation
is not activated.
