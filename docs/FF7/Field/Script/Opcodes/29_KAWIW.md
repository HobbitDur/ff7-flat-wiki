---
title: 29_KAWIW
---

-   Opcode: **0x29**
-   Short name: **KAWIW**
-   Long name: Wait For Kawai

#### Memory layout

| 0x29 |
|------|

#### Arguments

None.

#### Description

Halts script execution until a prior [KAWAI](28_KAWAI.md) operation has completed, such as a [LIGHT](28_KAWAI/06_LIGHT.md) movement.
