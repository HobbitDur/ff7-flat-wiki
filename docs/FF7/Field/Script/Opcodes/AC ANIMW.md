-   Opcode: **0xAC**
-   Short name: **ANIMW**
-   Long name: Wait for Animation

#### Memory layout

| 0xAC |
|------|

#### Arguments

None.

#### Description

Waits for the animation to complete that has been previously played
using any of the animation opcodes.
