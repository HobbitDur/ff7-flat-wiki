-   Opcode: **0x061**
-   Name: push greater
-   Two-word opcode: No

#### Parameters

| Location | Name |   Description    |
|:--------:|:----:|:----------------:|
|  Stack   |  A   | Value to compare |
|  Stack   |  B   | Value to compare |

#### Notes

Pushes A &gt; B to the stack.
