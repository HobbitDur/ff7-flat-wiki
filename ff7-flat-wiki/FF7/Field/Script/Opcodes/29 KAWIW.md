-   Opcode: **0x29**
-   Short name: **KAWIW**
-   Long name: Wait For Kawai

#### Memory layout

| 0x29 |
|------|

#### Arguments

None.

#### Description

Halts script execution until a prior [KAWAI][] operation has completed,
such as a [LIGHT][] movement.
