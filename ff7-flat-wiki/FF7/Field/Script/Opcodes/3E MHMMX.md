-   Opcode: **0x3E**
-   Short name: **MHMMX**
-   Long name: All HP/MP Maximum with Status Clear

#### Memory layout

| 0x3E |
|------|

#### Arguments

None.

#### Description

Restores full HP and MP of every available character, even those outside
the current party, as well as removing status effects.
