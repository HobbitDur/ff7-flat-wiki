-   Opcode: **0x65**
-   Short name: **SCRCC**
-   Long name: Scroll To Playable Character

#### Memory layout

| 0x65 |
|------|

#### Arguments

None.

#### Description

Instantaneously scrolls the current view so that it is centered on the
current playable character.
