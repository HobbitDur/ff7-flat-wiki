-   Opcode: **0xA4**
-   Short name: **VISI**
-   Long name: Visibility

#### Memory layout

| 0xA4 | *S* |
|------|-----|

#### Arguments

-   **const UByte** *S*: Switch on/off (1/0, respectively).

#### Description

Sets whether this entity's field object is visible.
