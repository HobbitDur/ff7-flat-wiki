-   Name: trigger battle
-   Two-word opcode: No

#### Parameters

| Location |     Name      |         Description         |
|:--------:|:-------------:|:---------------------------:|
|  Stack   | Battle scene? | The battle scene to trigger |

#### Notes

Immediately starts the specified battle, the format of the parameter has
not been verified.
