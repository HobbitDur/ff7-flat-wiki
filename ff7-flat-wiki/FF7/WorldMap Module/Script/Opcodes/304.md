-   Opcode: **0x304**
-   Name: set active entity direction & facing
-   Two-word opcode: No

#### Parameters

| Location |   Name    |       Description       |
|:--------:|:---------:|:-----------------------:|
|  Stack   | Direction | Value between 0 and 255 |

#### Notes

Sets the direction of movement and the facing of the current active
entity.
