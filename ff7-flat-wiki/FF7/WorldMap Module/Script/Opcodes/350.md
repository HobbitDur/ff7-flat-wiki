-   Opcode: **0x350**
-   Name: set meteor texture on/off
-   Two-word opcode: No

#### Parameters

| Location |  Name  |  Description  |
|:--------:|:------:|:-------------:|
|  Stack   | Meteor | Boolean value |

#### Notes

Turn the meteor in the sky overlay on/off.
