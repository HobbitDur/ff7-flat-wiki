-   Opcode: **0x047**
-   Short name: **MES**
-   Long name: Display Message

#### Argument

none

#### Stack

  
*Message channel*

*Message ID*

**MES**

#### Description

Popup a message window. This is usually used on lines to popup text when
the player crosses a certain point on a screen. The size of the message
window can be set with [WINSIZE][].
