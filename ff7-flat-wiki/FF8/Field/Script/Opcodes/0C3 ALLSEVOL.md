-   Opcode: **0x0C3**
-   Short name: **ALLSEVOL**
-   Long name: Set Volume of all Sound Effects

#### Argument

none

#### Stack

  
*Volume (0-127)*

**ALLSEVOL**

#### Description

Sets all playing sound effects to the given volume.
