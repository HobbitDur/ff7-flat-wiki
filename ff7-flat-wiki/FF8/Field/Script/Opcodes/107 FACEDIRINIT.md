-   Opcode: **0x107**
-   Short name: **FACEDIRINIT**
-   Long name: Initiate Head Facing

#### Argument

none

#### Stack

  
**FACEDIRINIT**

#### Description

Call this before using any other FACEDIR functions. It "unlocks" the
character's head from its model.
