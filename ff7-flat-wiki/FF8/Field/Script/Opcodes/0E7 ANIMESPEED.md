-   Opcode: **0x0E7**
-   Short name: **ANIMESPEED**
-   Long name: Set animation speed

#### Argument

none

#### Stack

  
*Speed (frames of animation per half second)*

**ANIMESPEED**

#### Description

Sets the speed of this entity's animations. Frame speed of 1 means 2
frames per second.
