-   Opcode: **0x02**
-   Short name: **JMP**
-   Long name: Jump

#### Argument

Number of instructions to jump (signed value).

#### Stack

No change.

#### Description

Jump a number of instructions given by **Argument**. If **Argument** is
negative, jumps backward.
