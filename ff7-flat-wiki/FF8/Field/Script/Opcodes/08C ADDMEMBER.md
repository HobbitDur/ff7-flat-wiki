-   Opcode: **0x08C**
-   Short name: **ADDMEMBER**
-   Long name: Add party member to game

#### Argument

none

#### Stack

  
*Player-Character ID*

**ADDMEMBER**

#### Description

Adds a PC to the available party (not to the active party).
