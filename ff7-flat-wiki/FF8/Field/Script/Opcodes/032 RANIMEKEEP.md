-   Opcode: **0x032**
-   Short name: **RANIMEKEEP**
-   Long name: Resume script, Animate, keep last frame

#### Argument

Model Animation ID

#### Stack

none

#### Description

Play an animation, then freeze the model on the last frame of the
animation.
