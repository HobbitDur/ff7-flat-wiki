-   Opcode: **0x08D**
-   Short name: **SUBMEMBER**
-   Long name: Remove party member from game

#### Argument

none

#### Stack

  
*Player-Character ID*

**SUBMEMBER**

#### Description

Removes a PC from the available party and the active party.
