-   Opcode: **0x02F**
-   Short name: **CANIME**
-   Long name: Controlled animation

#### Argument

Model Animation ID

#### Stack

  
*Last frame of animation*

*First frame of animation*

**CANIME**

#### Description

Play the range of frames of an animation, then return the model to its
base animation. Pauses script until animation finishes.
