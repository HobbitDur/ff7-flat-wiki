-   Opcode: **0x0FFF**
-   Short name: **SPECIAL: CLITM**
-   Long name: Special: Clear Items

#### Memory layout

| 0x0F | 0xFF |
|------|------|

#### Arguments

None.

#### Description

Removes all items from the party's inventory.
